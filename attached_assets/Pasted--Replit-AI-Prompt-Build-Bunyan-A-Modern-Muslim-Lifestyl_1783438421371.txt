# Replit AI Prompt — Build "Bunyan" (بنيان): A Modern Muslim Lifestyle Mobile App

You are a Senior React Native Engineer, Product Designer, UX Designer, Backend Architect, and Mobile Performance Expert.

Your task is to build a production-ready mobile application called **Bunyan (بنيان)**.

Do not create a simple prototype or UI mockup.

Build a clean, scalable, maintainable application that could realistically be published on both the Google Play Store and Apple App Store.

---

# App Name

**Bunyan (بنيان)**

### Tagline

**Build Yourself. Strengthen the Ummah.**

Inspired by the authentic hadith:

> "The believer to another believer is like a building whose different parts support one another."

The app is designed to help Muslim brothers become more disciplined in every aspect of life while encouraging accountability with trusted brothers.

This is NOT a social media application.

This is NOT a habit tracker.

This is a Personal Operating System for Muslim Men.
