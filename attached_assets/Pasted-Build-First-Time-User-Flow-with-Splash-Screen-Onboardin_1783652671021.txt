Build First-Time User Flow with Splash Screen & Onboarding

You are working on the existing Bunyan React Native (Expo + TypeScript) mobile application.

Do NOT redesign the app.

Keep the existing design system, theme, colors, typography, spacing, navigation, animations, Supabase integration, authentication, and folder structure.

Only implement the first-time user experience.

Objective

Implement a professional launch flow similar to Apple, Headspace, Duolingo, and Notion.

The app should intelligently determine whether the user is:

Opening the app for the first time
Returning after onboarding
Already authenticated
Not authenticated

After Splash Screen finishes:

isLoggedIn

Onboarding

isLoggedIn == true
